# 在线教育授课端项目 - 通用组件文档

## 一、表单类通用组件

### 1. 确认对话框（Confirm）
**路径**: `/src/components/common/form/confirm.vue`

**功能**: 提供标准的确认对话框，用于重要操作的二次确认。

**使用方法**:
```vue
<template>
  <confirm
    :title="'确认标题'"
    :content="'确认内容描述'"
    :show="dialogVisible"
    @confirm="handleConfirm"
    @cancel="handleCancel"
  />
</template>

<script setup lang="ts">
import Confirm from '@/components/common/form/confirm.vue';
import { ref } from 'vue';

const dialogVisible = ref(false);

const handleConfirm = () => {
  // 处理确认操作
  dialogVisible.value = false;
};

const handleCancel = () => {
  // 处理取消操作
  dialogVisible.value = false;
};
</script>
```

**属性**:
- `title`: 对话框标题
- `content`: 对话框内容
- `show`: 控制对话框显示/隐藏

**事件**:
- `confirm`: 点击确认按钮时触发
- `cancel`: 点击取消按钮时触发

### 2. 滑块组件（Slider）
**路径**: `/src/components/common/form/slider.vue`

**功能**: 提供范围选择滑块，适用于音量调节、亮度调节等场景。

**使用方法**:
```vue
<template>
  <tal-slider
    :slider-value="volume"
    :slider-width="200"
    :show-text="true"
    :enabled="!disabled"
    @slider="handleSliderChange"
  />
</template>

<script setup lang="ts">
import TalSlider from '@/components/common/form/slider.vue';
import { ref } from 'vue';

const volume = ref(50);
const disabled = ref(false);

const handleSliderChange = (value: number) => {
  volume.value = value;
  // 处理滑块值变化
};
</script>
```

**属性**:
- `sliderValue`: 当前滑块值（0-100）
- `sliderWidth`: 滑块宽度
- `showText`: 是否显示数值
- `enabled`: 是否启用

**事件**:
- `slider`: 滑块值变化时触发

**路径**: `/src/components/common/form/checkbox.vue`

```vue
import TalCheckbox from '@/components/common/form/checkbox.vue';
import { ref } from 'vue';

const isChecked = ref(false);
const isDisabled = ref(false);
const isHalfSelected = ref(false);

const handleChange = (value: boolean) => {
  isChecked.value = value;
  // 处理复选框状态变化

```

**属性**:
- `checkboxValue`: 是否选中
- `disabledValue`: 是否禁用
- `halfSelected`: 是否半选

**事件**:
- `changeStustus`: 状态变化时触发

## 二、消息提示类通用组件

### 1. 消息管理模块
**路径**: `/src/components/common/message/index.ts`

**功能**: 提供多种消息提示方法，包括确认对话框、Toast、提示信息等。

**使用方法**:
```javascript
import Message from '@/components/common/message/index';

// 确认对话框
Message.confirm({
  title: '确认标题',
  message: '确认内容',
  leftBtn: '取消',
  rightBtn: '确认',
  onSureCallBack: () => {
    // 确认回调
  },
  onCancelCallBack: () => {
    // 取消回调
  }
});

// 提示信息
Message.popTip('操作成功提示', 'parentId', 3000);

// Toast提示
Message.toast('加载成功', 2000, 'success');
Message.toast('操作失败', 2000, 'error');

// 弹出组件
Message.popup('弹出消息', 3000);
// 或弹出自定义组件
Message.popup(CustomComponent, 3000);
```

**API**:
- `confirm(options)`: 显示确认对话框
- `popTip(message, parentId, duration)`: 显示提示信息
- `toast(message, duration, type)`: 显示Toast提示
- `popup(message|component, duration, type)`: 显示弹出内容

### 2. 对话框组件（xDialog）
**路径**: `/src/components/common/message/xDialog`

**功能**: 提供可定制的对话框样式和交互。

### 3. Toast组件（xToast）
**路径**: `/src/components/common/message/xToast`

**功能**: 提供简洁的提示信息，自动消失。

## 三、其他通用组件

### 1. 加载中组件（Loading）
**路径**: `/src/components/common/loading/index.vue`

**功能**: 显示加载动画和提示文字，用于异步操作过程中。

**使用方法**:
```vue
<template>
  <div class="container">
    <loading :is-show="loading" :message="'加载中...'" />
  </div>
</template>

<script setup lang="ts">
import Loading from '@/components/common/loading/index.vue';
import { ref } from 'vue';

const loading = ref(true);

// 模拟加载完成
setTimeout(() => {
  loading.value = false;
}, 2000);
</script>
```

**属性**:
- `isShow`: 是否显示
- `message`: 提示文字（默认：加载中请稍后...）

### 2. 全局提示组件（GlobalTips）
**路径**: `/src/components/common/globalTips/index.vue`

**功能**: 显示在页面顶部的全局提示信息。

- 通过 `database.globalTips` 控制显示内容和状态

### 3. 音频播放器组件（AudioPlayer）
**路径**: `/src/components/common/audioPlayer/index.vue`

**功能**: 提供音频播放控制和事件监听。

**使用方法**:
```vue
<template>
  <div class="audio-container">
    <record-player ref="audioPlayerRef" :src="audioUrl" 
      @get-duration="handleGetDuration"
      @play-state="handlePlayState"
      @time-update="handleTimeUpdate"
      @buffer-state="handleBufferState"
    />
    <button @click="togglePlay">播放/暂停</button>
    <button @click="seekTo(10)">跳转到10秒</button>
  </div>
</template>

<script setup lang="ts">
import RecordPlayer from '@/components/common/audioPlayer/index.vue';
import { ref, onMounted } from 'vue';

const audioPlayerRef = ref<any>(null);
const audioUrl = 'https://example.com/audio.mp3';
const duration = ref(0);
const currentTime = ref(0);

onMounted(() => {
  // 组件挂载后可以访问方法
  audioPlayerRef.value.addEvent();
});

const togglePlay = () => {
  // 判断当前播放状态并切换
  if (/* 假设的播放状态检查 */) {
    audioPlayerRef.value.pause();
  } else {
    audioPlayerRef.value.play();
  }
};

const seekTo = (time: number) => {
  audioPlayerRef.value.seek(time);
};

const handleGetDuration = (value: number) => {
  duration.value = value;
};

const handlePlayState = (state: string) => {
  console.log('播放状态:', state);
};

const handleTimeUpdate = (time: number) => {
  currentTime.value = time;
};

const handleBufferState = (isBuffering: boolean) => {
  console.log('缓冲状态:', isBuffering);
};
</script>
```

**属性**:
- `src`: 音频文件URL

**事件**:
- `getDuration`: 获取音频时长
- `playState`: 播放状态变化
- `timeUpdate`: 播放时间更新
- `bufferState`: 缓冲状态变化

**方法**:
- `play()`: 播放音频
- `pause()`: 暂停音频
- `seek(time)`: 跳转到指定时间

## 四、通用组件使用最佳实践

### 1. 消息提示使用规范
- 重要操作使用 `confirm` 进行二次确认
- 操作结果反馈使用 `toast`
- 临时提示信息使用 `popTip`

### 2. 表单组件使用规范
- 统一使用组件库中的表单组件，保持样式一致性
- 根据功能需要选择合适的组件类型
- 表单状态管理推荐使用 Vue 的响应式系统

### 3. 性能优化建议
- 动态创建的组件（如消息提示）在使用完毕后及时销毁
- 避免在组件中添加过多不必要的事件监听器
- 对于频繁使用的组件，考虑使用缓存策略

### 4. 样式一致性维护
- 通用组件的样式统一管理在 `/src/components/common/form/index.less` 等文件中
- 组件样式应遵循项目的设计规范
- 使用 CSS 变量或 LESS 变量管理主题颜色

## 五、注意事项

1. 组件导入路径使用 `@` 别名指向 `src` 目录
2. 部分组件使用了 Options API，部分使用了 Composition API，注意兼容使用
3. 消息相关组件会自动处理组件的创建和销毁，无需手动管理生命周期
4. 使用数据库（database）管理全局状态的组件，需要正确初始化数据库模块
