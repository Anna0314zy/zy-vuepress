# 三、进阶必看 4 个资源

## 1 CLAUDE.md 模板

几乎所有高手都写这个文件。

典型内容：

```text
project architecture
coding rules
test rules
folder structure
```

示例：

```md
## Coding rules

- Use TypeScript
- Use composition API
- No any types
```

Claude Code 会按照规则改代码。

---

## 2 hooks 示例

Hooks 可以自动执行操作。

例如：

```text
run lint
run test
format code
```

很多团队用：

```text
Claude Code 自动修 lint
```
